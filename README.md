# Roxiler Store Rating System

## Overview
A Full Stack Store Rating Platform built using React, Node.js, Express.js and MySQL.

## Features

- User Authentication
- Role Based Access (Admin, Owner, User)
- Store Management
- User Management
- Rating System
- Dashboard Statistics
- Search Stores

## Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Framer Motion

### Backend
- Node.js
- Express.js

### Database
- MySQL
